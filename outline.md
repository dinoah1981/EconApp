# Scarcity Simulation - Project Outline

## File Structure

### Core HTML Pages
1. **index.html** - Main landing page with simulation selection
2. **pizza-parlor.html** - Pizza restaurant resource allocation game
3. **time-management.html** - 24-hour time allocation challenge
4. **budget-game.html** - Weekly budget constraint simulation
5. **island-survival.html** - Resource management on deserted island
6. **learn-more.html** - Educational content and real-world examples

### Supporting Files
- **main.js** - Core JavaScript functionality and shared utilities
- **resources/** - Image assets, icons, and media files
- **styles/** - CSS files for styling (embedded in HTML)

## Page Content Breakdown

### Index.html - Landing Page
- Hero section with animated introduction to scarcity
- Interactive simulation selector with preview cards
- Brief explanation of what students will learn
- Navigation to different scenarios
- Achievement showcase area

### Pizza Parlor Simulation
- **Objective**: Manage a pizza restaurant with limited ingredients, time, and money
- **Decisions**: Ingredient purchases, menu pricing, staff allocation
- **Learning**: Opportunity cost, resource allocation, profit maximization
- **Duration**: 5-7 minutes per session

### Time Management Challenge
- **Objective**: Allocate 24 hours across different activities
- **Decisions**: Sleep, homework, sports, social time, entertainment
- **Learning**: Time as a scarce resource, trade-offs, prioritization
- **Duration**: 3-5 minutes per session

### Budget Game
- **Objective**: Spend weekly allowance across different categories
- **Decisions**: Food, entertainment, savings, clothes, transportation
- **Learning**: Budget constraints, opportunity cost, financial planning
- **Duration**: 4-6 minutes per session

### Island Survival
- **Objective**: Survive on island with limited resources
- **Decisions**: Shelter, food, water, tools, exploration
- **Learning**: Resource scarcity, cooperation, survival economics
- **Duration**: 6-8 minutes per session

### Learn More Page
- Real-world examples of scarcity
- Historical case studies
- Economic concepts explained simply
- Discussion questions for classroom use
- Additional resources and links
