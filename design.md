# Scarcity Simulation - Design System

## Design Philosophy

### Color Palette
- **Primary**: Warm coral (#FF6B6B) - energetic and engaging for teens
- **Secondary**: Deep teal (#4ECDC4) - trustworthy and calming
- **Accent**: Golden yellow (#FFE66D) - optimistic and attention-grabbing
- **Neutral**: Charcoal gray (#2C3E50) - professional and readable
- **Background**: Soft cream (#FEFEFE) - clean and accessible

### Typography
- **Display Font**: "Poppins" - modern, friendly, highly readable
- **Body Font**: "Inter" - clean, professional, excellent for educational content
- **Accent Font**: "Fredoka One" - playful for headings and special elements

### Visual Language
- Clean, modern interface with plenty of white space
- Bold, colorful illustrations and icons
- Subtle shadows and rounded corners for a friendly feel
- Clear visual hierarchy with strong contrast
- Interactive elements that feel responsive and alive

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth animations for UI transitions and feedback
- **p5.js**: Interactive visualizations and creative coding elements
- **ECharts.js**: Data visualization for simulation results
- **Splide.js**: Image carousels and content sliders

### Animation Effects
- Gentle fade-ins for content sections
- Bouncy feedback for button interactions
- Smooth progress bars for simulation states
- Particle effects for achievement celebrations

### Header Effect
- Animated gradient background using CSS and Anime.js
- Floating geometric shapes that respond to mouse movement
- Dynamic color transitions that reflect simulation outcomes

### Interactive Elements
- Hover effects with subtle scale and shadow changes
- Color-coded feedback for different types of decisions
- Visual progress indicators throughout simulations
- Achievement badges with celebration animations

### Responsive Design
- Mobile-first approach with touch-friendly interactions
- Flexible grid system that adapts to different screen sizes
- Optimized font sizes and spacing for readability
- Accessible color contrast ratios throughout
